Пакетный GPS импортер

Вводу и вывод

Закладка Ввод и вывод содержит всю обязательную информацию необходимую для создания слоя из gpx-файлов. Изображение ниже иллюстрирует все поля из закладки. На изображении, поля отмеченные звездочки обязательны для заполнения.

Ниже приводится описания каждого из полей.

Заметка: Вы можете использовать примеры расположенные в папка плагина (обычно .qgis\python\plugins\batch_gpx_importer) -> sample_data.

Папка GPX

Выберите папку расположения gpx-файлов нажав кнопку Обзор...

Префикс имени файла

Позволяет искать только названия файлов начинающися с определённых символов.

Постфикс имени файла

Позволяет искать только имена файлов заканчивающиеся на определённые символы.

Сканировать подпапки

Выберите Да для импортирования файлов также из подпапок или Нет для импорта файлов только из текущей папки.

Формат GPX

Выберите формат GPX из трэков, маршрутных точек или путей. Формат учитывается при сборе данных GPX.

Имя слоя

Устанавливает имя результирующего слоя. Если не указано, используется слой по умолчанию.

Тип геометрии

Выберите тип выходной геометрии в результирующем слое. Может быть Полигоном, Линией, Точкой. Если будет выбран Полигон или Линия, будут собираться лишь атрибуты лишь первого объекта. Для Точки будут собраны все атрибуты.

Проекция

Выберите координатную систему используемую при сборе GPS точек. Также устанавливает проекцию результирующего слоя.

Проверка

Закладка Проверка позволяет фильтровать объекты по заданным правилам и екстенту карты. Также позволяет дополнительно сохранять файлы которые были успешно импортированы и которые были отфильтрованы. Изображение снизу иллюстрируют все поля закладки Проверка.

Ниже приводится описания каждого из полей.

Исключить за границами экстента

При активации границ экстента автоматически захватывается экстент карты. При нажатии на клавишу Импортировать, этот экстент будет использоваться для обрезания объектов за пределами видимости карты.

Исключить объекты с ошибкой геометрии.

Выберите сохранить или удалить объекты с ошибками геометрии. Если выберите Да, объекты с ошибками геометрии будут исключены, информация о них и gpx-файле будет отображена в Логе.

Исключить объекты с недостатком вершин

Включить или исключить объекты с недостающими для построения верной геометрии вершинами. Этот фильтр работает для линий и полигонов. Для линии, минимальное количество необходимых вершин - 2. Для полигона - 3.

Если вы выберите Да, объекты с количеством вершин меньше необходимых будут исключены. Если вы выберите Нет, объекты с количеством вершин меньше необходимых будут импортированы. Это приведет к наличию в слое объектов без видимой геометрии.

Папка отфильтрованных GPX

Выберите папку в которой хотите сохранить успешно импортированные gpx-файлы. Если значение не установлено, файлы не будут скопированы.

Папка импортированных GPX

Выберите папку в которой хотите сохранить отфильтрованные или неудачные gpx файлы не попавшие в результирующий слой. Если значение не установлено, файлы не будут скопированы.

Поля

Закладка Поля содержит опции выбора необходимых полей которые будут скопированы в результирующий слой из gpx-файла, а также дополнительные поля, создаваемые самим плагином.

Выберите поля слоя

Вы можете включить или исключить поля которые будут добавлены в результирующий слой карты. Если вы снимите отметку с поля, оно не будет включено в результирующий слой, а информация в нём будет потеряна. Изображение ниже показывает, как включать и исключать поля.

Поле id будет добавлено по умолчанию и не может быть исключено.

Все поля за исключением feature_type(тип объекта) and file_name(имя исходного файла) будут получены из gpx-файлов.

Процесс импорта

Как только вы заполнили все необходимые поля, вы можете начать процесс импорта gpx-объектов нажав кнопку Импортировать, как изображено ниже.

Как только вы нажмете кнопку Ипортировать, появится окно прогресса и закладка Лог, показывающие ход процесса как показано ниже.

После завершения импорта, вы увидите сообщение об успешном импорте, а в проект будет добавлен новый результирующий слой, как показано ниже.

Динамическая помощь

Пакетный GPS Импортер имеет функцию динамической помощи при которой вы можете увидеть подробное описание каждого поля. Для активации, нажмите кнопку Показать динамическую помощь расположенную внизу окна. Окно динамической помощи появится с правой стороны диалога, как показано ниже.

Кликните по заголовку каждого поля, что бы увидеть подробное описание, как показано ниже.
